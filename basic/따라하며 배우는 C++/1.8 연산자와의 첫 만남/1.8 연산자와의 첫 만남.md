# 1.8 연산자와의 첫 만남

- 리터럴(Literal): 코드에 직접 쓴 고정된 값
    - `int a = 10;`
    - `double b = 3.14;`
    - `char c = 'A';`
    - `bool d = true;`
- 피연산자(Operand): 연산자(operator)가 연산을 수행하는 대상
    - `int result = 3(피연산자) + 5(피연산자);`
- 단항(unary): 피연산자가 1개
    ```cpp
    int a = 5;

    -a    // 부호 반전 -> -5
    !true // 논리 NOT -> false
    ++a   // 증가 연산자 -> 6
    --a   // 감소 연산자 -> 5
    ```
- 이항(binary): 피연산자가 2개
    ```cpp
    3 + 5
    10 - 2
    4 * 3
    9 / 3
    10 % 3
    a = b // 대입
    ```
- 삼항(ternary): 피연산자가 3개
    - C++ 에서 유일한 삼항 연산자는 `? :`
    - if-else 를 한 줄로 축약
    ```cpp
    int a = 10;
    int b = (a > 5) ? 100 : 200;
    ```
