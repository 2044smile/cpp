# 3.4 sizeof, 쉼표 연산자, 조건부 연산자

## sizeof 연산자(operator)란

> 자로형 또는 변수의 메모리 크기(bytes)를 반환한다
> 데이터형의 크기를 알고 싶을 때 사용한다
> 메모리 관리에 필수적으로 사용된다

- sizeof(타입명), sizeof(변수명) 두 가지 방식으로 사용 가능
- 런타임이 아닌 컴파일 타임에 크기가 결정된다 -> 성능 부담 없음
- 메모리 관리, 배열 크기 계산, 플랫폼 간 이식성 확보에 사용

## 쉼표 연산자 (Comma Operator)

> 왼쪽부터 순서대로 표현식을 실행하고, 마지막 표현식의 결과만 반환한다

```cpp
int x = 3, y = 10;
int z = (++x, +=y); // x=4 로 증가하지만 버려짐, y=11이 z에 저장
cout << z << endl;  // 11
```

### = 와 ,의 우선순위 주의

- 쉼표 연산자는 C++에서 우선순위가 가장 낮은 연산자다

```cpp
z = a, b;   // (z = a), b -> z = a만 저장되고 b는 버려짐 (경고 발생)
z = (a, b); // z = (a, b) -> b가 z에 저장됨

int a1 = 1, b1 = 10;
int z1 = (++a1, a1 + b1);   // a1=2로 증가 후,  2 + 10 = 12가 z1에 저장
```

### 현업에서 자주 쓰는 패턴

```cpp
// 직접적인 사용은 지양한다. 가독성을 해치지 때문 for 루프에서는 관용적으로 사용한다
for (int i = 0, j = 10; i < j; ++i, --j) { }
```

## 조건부 연산자 

> 조건이 참이만 앞의 값, 거짓이면 뒤의 값을 반환한다

```cpp
const in price = (onSale == ture) ? 10 : 100;
```

### if문과의 핵심 차이

| 구분        | if문      | 삼항 연산자      |
| --------- | -------- | ----------- |
| 역할        | 실행 흐름 제어 | 값을 반환하는 표현식 |
| const 초기화 | 불가       | 가능          |

```cpp
// if문 - const 불가
const int price;
if (onSale) price = 10; // 에러: const는 선언과 동시에 초기화해야 함

// 삼항 연산자 - const 가능
const int price = (onSale) ? 10 : 100;

// 함수로 분리하는 것도 좋은 방법
const int price = getPrice(onSale);
```

### 주의사항

#### 괄호 처리: `<<` 연산자와 우선순위 충돌 방지를 위해 괄호 필요

```cpp
cout << ((x % 2 == 0 ? "even" : "odd") << endl;)
```

#### 타입 불일치: 참/거짓의 반환 타입이 달라지면 컴파일러가 암묵적 변환을 시도해 예기치 않은 동작이 발생할 수 있다

```cpp
cout << ((x % 2 ==0) ? 0 : "odd") << endl;
```
