# 3.8 비트단위 연산자 Bitwise Operators

> 메모리 주소 구조상 bytes 단위
> 비트단위 연산자를 사용하면 속도가 빠르다
> unsigned를 사용

## 연산자

- << left shift operator : 비트 단위로 왼쪽으로 이동 eg. 0011 -> 0110
    - 2를 곱하는 연산을 빠르게 하는 비트 연산
- >> right shift operator : 비트 단위로 오른쪽으로 이동
- ~ bitwise NOT operator : 비트 단위로 NOT 연산
- & bitwise AND operator : 비트 단위로 AND 연산
- | bitwise OR operator : 비트 단위로 OR 연산
- ^ bitwise XOR operator : 비트 단위로 XOR 연산
