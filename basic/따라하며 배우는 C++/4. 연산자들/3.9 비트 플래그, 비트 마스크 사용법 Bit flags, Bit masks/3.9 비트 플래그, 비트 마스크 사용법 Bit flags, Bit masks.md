# 3.9 비트 플래그, 비트 마스크 사용법 Bit flags, Bit masks

## 기본 개념

- 비트 플래그와 마스크는 하나의 정수형 변수의 각 비트를 사용하여 여러 개의 독립적인 참/거짓 상태를 표현하는 기법입니다.
    - 이를 통해 메모리를 절약하고 상태 정보를 효율적으로 관리할 수 있어요.
- 비트 플래그 = 각 비트를 ON/OFF 스위치로 사용하는 것
- 8비트면 스위치 8개를 변수 하나로 관리할 수 있다

```txt
00000000
↑↑↑↑↑↑↑↑
│││││││└── item0 (opt0)
││││││└─── item1 (opt1)
│││││└──── item2 (opt2)
└└└└└───── ...
```

## 1. 플래그 정의 `<<` (비트 시프트)

- `1 << n` = 1을 왼쪽으로 n칸 밀기

```cpp
const unsigned char opt0 = 1 << 0;
const unsigned char opt1 = 1 << 1;
const unsigned char opt2 = 1 << 2;
const unsigned char opt3 = 1 << 3;

// opt0 1 << 0  →  00000001  (1을 0칸 밀기)
// opt1 1 << 1  →  00000010  (1을 1칸 밀기)
// opt2 1 << 2  →  00000100  (1을 2칸 밀기)
// opt3 1 << 3  →  00001000  (1을 3칸 밀기)
```

## 2. 플래그 켜기 `|=` (OR 연산)

```cpp
items_flag != opt0; // 00000001
items_flag != opt1; // 00000011
items_flag != opt3; // 00001011
```

## 3. 플래그 끄기 `&= ~` (AND + NOT 연산)

- `~opt3` -> 비트 전부 뒤집기 (NOT)
- `&=` -> AND 연산

### 1단계

- `~opt3` -> 비트 전부 뒤집기 (NOT)

```txt
opt3  = 00001000
~opt3 = 11110111  ← 끄고 싶은 비트만 0, 나머지는 1
```

### 2단계

- `&=`` -> AND 연산
    - AND 연산 규칙: 둘 다 1일 때만 1

```txt
items_flag = 00001011
~opt3      = 11110111
↓ AND 연산 (둘 다 1일 때만 1)
           = 00000011  ← item3만 꺼짐, 나머지 유지
```

## 4. 플래그 확인 `&` (AND 연산)

```cpp
if (items_flag & opt1) { cout << "Has item1" << endl; }
if (items_flag & opt0) { cout << "Has item0" << endl; }
```

```txt
items_flag = 00000011  (item0, item1 켜진 상태)

& opt1 (00000010)
00000011
00000010
--------
00000010  ← 0이 아님 → true → "Has item1" 출력

& opt0 (00000001)
00000011
00000001
--------
00000001  ← 0이 아님 → true → "Has item0" 출력
```
