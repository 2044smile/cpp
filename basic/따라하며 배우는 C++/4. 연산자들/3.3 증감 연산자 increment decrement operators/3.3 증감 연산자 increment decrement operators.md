# 3.3 증감 연산자 increment decrement operators

> 많이 쓰이고 쉬운데 실수가 많이 발생한다

## ++x (prefix), x++ (postfix)의 사용 이유와 실제 상황

### 반복문에서 (가장 흔한 사용)

```cpp
for (int i = 0; i < 10; i++)
{
    cout << i << endl;
}
for (int i = 0; i < 10; ++i)
{
    cout << i << endl;
}
```

### 데이터 저장 순서가 중요한 경우

```cpp
int arr[5] = {0};
int i = 0;

// 후위 연산자 (x++)
arr[i++] = 10;
arr[i++] = 20;
arr[i++] = 30;
```

### 실제 코드

```cpp
// 큐(Queue) 자료구조
queue<int> q;
q.push(15);
q.push(20);

int front_val = q.front();
q.pop()

// 카운터 증가
int cout = 0;
cout++  // 단순히 카운트만 증가 (값 사용 안 함, 효율적)
int next = ++count
```
