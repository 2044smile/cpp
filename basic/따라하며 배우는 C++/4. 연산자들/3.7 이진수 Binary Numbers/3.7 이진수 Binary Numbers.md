## signed vs unsigned

> 범위가 다르다 (같은 비트, 다른 해석)

```cpp
// 비트는 1001 1110 똑같다
unsigned char a = 158;  // 0~255 정수
signed char b   = -98   // -128~127 음수~정수

// 1001 1110
// ↓
// unsigned -> 그냥 계산
// 2^7 + 2^4 + 2^3 + 2^2 + 2^1
// = 128 + 16 + 8 + 4 + 2
// = 158

// signed -> 맨 앞 1 = 음수 -> 뒤집고 +1
// 1001 1110 -> 0110 0001 -> 0110 0010 = 98 = -98
```
