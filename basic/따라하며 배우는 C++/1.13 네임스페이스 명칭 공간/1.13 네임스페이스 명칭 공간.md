# 1.13 네임스페이스 명칭 공간

## 구조

- 코드에서 두 개의 namespace가 있다
    - `tSpace::doSomething`
    - `tSpace::innerSpace::func`

## using namespace 역할

```cpp
using namespace tSpace;
using namespace tSpace::innerSpace;

// using 덕분에 아래와 같이 구현 할 수 있다
doSomething(3,4);
func(3,4);
```

## using namespace 단점

- 헤더파일에서 절대 사용하지 않는다

## modern C++(17) 스타일

- nested namespace를 한 줄로 표현

```cpp
namespace tSpace::innerSpace
{
    int func(int a, int b)
    {
        return a * b
    }
}
```

## AWESOME CODE (namespace + alias)

```cpp
#include <iostream>

namespace math
{
    int add(int a, int b)
    {
        return a + b;
    }
    int multiply(int a, int b)
    {
        return a * b;
    }
}

namespace m = math;
using namespace std;

int main()
{
    cout << m::add(3,4) << endl;
    cout << m::multiply(3,4) << endl;
}
```
