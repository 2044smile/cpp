# 1.3 변수와의 첫 만남

## 객체(objects)

- OOP 객체지향 프로그래밍
- 프로그램 실행 중 메모리에 생성되어 저장되는 데이터 단위

## 변수(variables)

- 메모리에 있는 객체를 사용하기 위해서는 이름이 필요하다
- 변수는 메모리에 저장된 객체를 가리킨다
- 실제로는 0x7ffab123 하지만 이렇게 쓰기 어렵기에 이 주소에 이름을 붙여서 사용하는 것이 변수

## Left-values 과 Right-values

### Left-values

> 메모리 주소를 가지며 이름을 가진 객체

- 메모리에 실제로 존재
- 주소(&)를 얻을 수 있음
- 대입문의 왼쪽에 올 수 있음

```cpp
int a = 10;
std::cout << &a; // a는 메모리 주소가 존재하기 때문에 `&a` 가능
```

#### reference

```cpp
int a = 10;
int& ref = a;  // ref -> a를 참조

// ref = 20; 불가능 ref는 다른 걸 가리킬 수 없음
```

### Right-values

> 일시적인 값(temporary value)

- 이름이 없음
- 주소를 직접 사용할 수 없음
- 보통 연산 결과

```cpp
int a = 10 + 20;
```
