# 4.1 지역 변수, 범위(Scope), 지속기간(Duration)

## 1. Scope(범위)

> 변수는 선언된 `{}` 블록 안에서만 유효하다.

```cpp
int main(){
    int apple = 5;
    {
        apple = 1;      // 외부 변수 수정 가능
        cout << apple;  // 1
    }
    cout << apple;      // 1 (블록 안에서 수정된 값 유지)
}
// apple = 1;           // ❌ main() 밖에서는 접근 불가
```

## 2. namespace(네임스페이스)

> 같은 이름의 변수/함수가 충돌하지 않도록 영역을 구분하는 것

```cpp
namespace work1 {
    int a = 1;
    int doSomething() {
        a += 3;
        return a;
    }
}

namespace work2 {
    int a = 1;              // work1::a 와 충돌 없음
    int doSomething() {     // work1::doSomething() 과 충돌 없음
        a += 5;
        return a;
    }
}
```

## 3. 중첩 네임스페이스 (C++17)

> `::`로 네임스페이스를 중첩할 수 있다

```cpp
// C++17 이전
namespace work3 {
    namespace work33 {
        namespace work333 {
            int a = 1;
        }
    }
}

// C++17 이후 (간결하게 가능)
namespace work3::work33::work333 {
    int a = 1;
    int doSomething() {
        a += 3;
        return a;
    }
}

// 사용
work3::work33::work333::doSomething();
```

## 4. 함수 반환타입 주의

|반환타입|return 필요 여부|cout 출력 가능|
|--|--|--|
|`void`|❌ 없어도 됨|❌ 불가|
|`int`|✅ 반드시 필요|✅ 가능|

```cpp
// ❌ 잘못된 예
int doSomething() {
    a += 3;
    // return 없음 → 경고 발생
}

// ✅ 올바른 예
int doSomething() {
    a += 3;
    return a;
}
```

## 6. & vs && / | vs ||

|연산자|종류|용도|
|--|--|--|
|`&`|비트 AND|비트끼리 연산|
|`&&`|논리 AND|조건이 둘 다 참인지 확인|
|`\|`|비트 OR|비트끼리 연산|
|`\|\|`|논리 OR|조건 중 하나라도 참인지 확인|

```cpp
// ✅ 조건문에서는 논리 연산자 사용
if (a % 2 == 1 && b % 2 == 1) { ... }
if (a % 2 == 0 || b % 2 == 0) { ... }
```
