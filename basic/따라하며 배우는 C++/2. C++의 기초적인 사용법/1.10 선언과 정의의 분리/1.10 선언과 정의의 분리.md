# 1.10 선언과 정의의 분리

## 컴파일러

- line 1 부터 순서대로 읽는다
- main() 함수가 위에 선언되어 있고, 다른 함수를 main() 함수 아래 선언하고 실행하면 에러 발생

## forward declaration(전진 선언)

```cpp
#include <iostream>

using namespace std;

int add(int a, int b); // *
int multiply(int a, int b); // *
int subtract(int a, int b); // *

int main()
{
    cout << add(1, 2) << endl;
    cout << multiply(3, 4) << endl;
    cout << subtract(5, 6) << endl;

    return 0;
}

// definition
int add(int a, int b){
    return a + b;
}
int multiply(int a, int b)
{
    return a * b;
}
int subtract(int a, int b)
{
    return a - b;
}
```
