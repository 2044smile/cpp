# 1.8 연산자와의 첫 만남

- 리터럴(Literal): 코드에 직접 쓴 고정된 값
    - `int a = 10;`
    - `double b = 3.14;`
    - `char c = 'A';`
    - `bool d = true;`
- 피연산자(Operand): 연산자(operator)가 연산을 수행하는 대상
    - `int result = 3(피연산자) + 5(피연산자);`
- 단항(unary): 피연산자가 1개
    ```cpp
    int a = 5;

    -a    // 부호 반전 -> -5
    !true // 논리 NOT -> false
    ++a   // 증가 연산자 -> 6
    --a   // 감소 연산자 -> 5
    ```
- 이항(binary): 피연산자가 2개
    ```cpp
    3 + 5
    10 - 2
    4 * 3
    9 / 3
    10 % 3
    a = b // 대입
    ```
- 삼항(ternary): 피연산자가 3개
    - C++ 에서 유일한 삼항 연산자는 `int result = (a > b) ? a : b`
    - if-else 를 한 줄로 축약
    ```cpp
    int a = 10;
    int b = (a > 5) ? 100 : 200;
    ```

## 심화

- C++에서는 연산자 오버로딩(Operator Overloading)이 가능합니다.
- Vector 클래스에 + 연산자를 정의하면 v1 + v2 문법이 동작한다.

```cpp
#include <iostream>
using namespace std;

class Vector {  // Vector라는 사용자 정의 자료형
public:         // public 접근 지정자: 클래스 바깥에서도 사용할 수 있게 공개(public)
    int x, y;   // 멤버 변수: Vector가 가지는 데이터

    Vector(int x, int y) : x(x), y(y) {}    // 생성자(Constructor): 객체를 만들 때 자동으로 호출

    // + 연산자 오버로딩: + 연산자를 직접 정의한다
    Vector operator+(const Vector& other) { // Vector&: 복사 방지
        return Vector(x + other.x, y + other.y);
    }
};

int main() {
    Vector v1(1, 2);
    Vector v2(3, 4);

    Vector v3 = v1 + v2;    // mean v1.opterator+(v2);

    cout << v3.x << " " << v3.y << endl; // 4 6
}
```
