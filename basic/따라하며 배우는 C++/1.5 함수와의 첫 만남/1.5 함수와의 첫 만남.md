# 1.5 함수와의 첫 만남

## 설명

### 함수의 선언

- 함수 선언과 정의를 분리하기도 한다
    - 함수 선언 : 함수의 이름과 매개변수, 반환 타입을 알려주는 것
    ```cpp
    // main 위쪽
    int multiplyTwoNumbers(int num_a, int num_b)
    ```
    - 함수 정의 : 함수의 실제 구현, 함수 선언과 정의를 합쳐서 작성하기
    ```cpp
    // main 아래쪽
    int multiplyTwoNumbers(int num_a, int num_b)
    {
        return num_a * num_b;
    }
    ```
    - 예
    ```cpp
    #include <iostream>
    using namespace std;

    int multiplyTwoNumbers(int num_a, int num_b);

    int main()
    {
        cout << multiplyTwoNumbers(3,4);
        // 줄바꿈이 필요하면 endl 보다 "\n"
        cout << multiplyTwoNumbers(3,4) << "\n";
    }

    int multiplyTwoNumbers(int num_a, int num_b)
    {
        return num_a * num_b;
    }
    ```

- 함수 선언 할 때 출력 값의 데이터 타입 == 반환 타입
    - 매개변수(Parameter) : 함수 정의할 때 변수
    - 인자(Argument) : 함수 호출할 때 값

### 변수의 범위 (Scope)

```cpp
int multiplyTwoNumbers(int num_a, int num_b)
{
    int result = num_a * num_b;
    return result; // result는 함수 안에서만 존재 == Local Variable
}
```

### 함수 네이밍

- 좋은 함수 이름 특징
    - 동사로 시작
    - 무엇을 하는지 명확
