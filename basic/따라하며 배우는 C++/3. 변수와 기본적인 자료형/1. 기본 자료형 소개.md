# 1. 기본 자료형 소개

## 문자열이 어떻게 컴퓨터에 저장이 될까?

```cpp
char a = 'H'; // 문자는 1 byte
// 'H' -> ASCII -> 72 -> 이진수 -> 01001000
int i = 1; // 정수는 4 byte == 32bits
// 00000000 00000000 00000000 00000001
```

- 주소는 계속 변경된다
- 메모리에 저장되는 사이즈가 다르다

## Group

### Character types

- char: 8bits
- char16_t: 16bits
- char32_t: 32bits
- wchar_t: Can represent(대표하다) the largest supported character set

### Integer types (signed)

- signed char: 8bits
- signed short int: 16bits
- signed int: 16bits
- signed long int: 32bits
- signed long long int: 64bits
