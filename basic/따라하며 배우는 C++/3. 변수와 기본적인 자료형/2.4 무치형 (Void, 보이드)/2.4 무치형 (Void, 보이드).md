# 2.4 무치형 (Void, 보이드)

> Void Type는 메모리 차지 안함

## void = "없음"

> `void`는 말 그대로 "아무것도 없다" 는 뜻

```cpp
int     i;  // 4byte 공간 필요
float   f;  // 4byte 공간 필요
char    c;  // 1byte 공간 필요

void    v;  // 불가!
```

## void 가 쓰이는 곳

### 1. 함수 반환값이 없을 때

```cpp
int main() {
    return 0;
}

// void 는 반환값 없음 -> return 안써도 됨
void myFunction() {
    cout << "그냥 실행만 하고 끝";
    // return 필요 없음
}
```

### 2. void* 포인터란 (타입 없는 주소)

> 일반 포인터는 타입이 고정되어 있지만, `void*`는 "타입 상관없이 아무 주소나 저장 가능" 한 포인터
> `void*`는 타입 상관없이 아무 주소나 저장할 수 있지만 값을 꺼낼 때는 타입을 알려줘야 합니다

```cpp
int     i = 123;
float   f = 123.456f;

void* my_void;

my_void = (void*)&i;    // int 주소 저장
my_void = (void*)&f;    // float 주소 저장
```

## 포인터란

> 포인터는 변수의 주소를 저장

### 핵심 개념: "주소"

- 변수: 집 (실제 데이터가 사는 곳)
- 주소: 집 주소 (어디 있는지 알려주는 것)
- 포인터: 주소를 적어놓는 메모지 

### 메모리를 실제로 상상해보기

- 변수를 선언하면 메모리 어딘가 공간이 잡히고, 그 공간의 `주소`가 생긴다

```txt
메모리 (RAM) 구조:

주소     값
0x0001 [ 123     ]  <- int i = 123
0x0002 [         ]
0x0003 [         ]
0x0004 [         ]
0x0005 [ 123.456 ]  <- float f = 123.456f
```

### &와 *의 차이

```cpp
int i = 123;

&i  // i 의 "주소"를 알려줘 (& = address of)
    // 0x0001 같은 숫자가 나옴

int* p = &i  // 포인터 p에 i의 주소를 저장
*p;     // p가 가리키는 주소에 있는 "값"을 알려줘 (* = dereference)
        // 123이 나옴
```
