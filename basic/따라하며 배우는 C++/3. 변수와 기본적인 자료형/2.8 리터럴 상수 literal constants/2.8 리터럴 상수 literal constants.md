# 2.8 리터럴 상수 literal constants

> 상수 constants

## 리터럴 상수

- 리터럴 상수(Literal Constant)는 소스 코드에 직접 작성된 고정된 값입니다. 
- 변수에 저장되지 않고, 프로그램이 실행될 때 메모리에 직접 포함되는 상수값입니다.

### 예제
```cpp
float pi = 3.14;  // 3.14는 리터럴 상수
int age = 25;     // 25는 리터럴 상수
```

### 리터럴 상수의 종류

| 종류 | 예제 | 설명 |
|------|------|------|
| **정수형 리터럴** | `10`, `0xFF`, `0b1010` | 10진, 16진, 2진 정수 |
| **부동소수점 리터럴** | `3.14`, `2.5f`, `1.0e-3` | 실수 (float, double) |
| **문자형 리터럴** | `'A'`, `'\n'` | 단일 문자 |
| **문자열 리터럴** | `"Hello"` | 문자열 |
| **불리언 리터럴** | `true`, `false` | 참/거짓 |

### 리터럴 상수 vs. 변수
- **리터럴**: 값이 고정되고 코드에 직접 작성됨
- **변수**: 이름으로 값에 접근하고, 값을 변경할 수 있음

```cpp
int x = 5;     // 5는 리터럴 상수, x는 변수
x = 10;        // 변수 x의 값을 변경 가능
// 10은 새로운 리터럴 상수
```

## 2, 8, 16진수를 코딩할 때 응근 많이 사용한다

- `0011 1010 1111 1111(2진수) -> 3A7F(16진수)`

### 2진수

```cpp
int x = 0b1010;
cout << x << endl;  // 10
```
