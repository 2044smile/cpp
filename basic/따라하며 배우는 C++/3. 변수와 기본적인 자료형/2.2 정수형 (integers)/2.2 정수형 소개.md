# 2.2 정수형 소개

> 일반적인 경우에는 무조건 int를 사용한다
> 메모리 사이즈가 제한이 되어 있다
> 표현할 수 있는 숫자가 제한이 되어 있다
> 부호가 있고 없고에 따라 차이가 있을 수 있다
> overflow, underflow

|영역(Category)|형(Type)|최소 크기|기타|
|--|--|--|--|
|문자(character)|char|1 바이트|
|정수(integer)|short|2바이트|
||int|2바이트|대부분 4바이트|
||long|4바이트||
||long long|8바이트|C99/C++11 type|

## 정수 데이터 구조

```cpp
int i = 1;  // 32bits - 4byte
// 00000000 00000000 00000000 00000001
```
