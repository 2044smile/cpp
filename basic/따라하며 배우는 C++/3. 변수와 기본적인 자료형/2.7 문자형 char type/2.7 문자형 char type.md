## BUFFER

- 입력 버퍼는 OS + C++ 스트림이 관리한다 우리가 메모리처럼 free/delete 할 대상이 아니다
- 버퍼 메모리는 프로그램 실행 중 유지되고, 입력 소비되면 내부적으로 재사용되고, 프로그램 종료되면 OS가 정리한다
- 데이터가 들어올 때 한꺼번에 처리를 못하면 버퍼라는 곳에 저장해둔다
- 버퍼는 자동으로 사라지지 않는다 (가비지 컬렉션(GC) 없음)
    - 버퍼 비우는 방법
        1. ignore(): `cin.ignore(1000, '\n');` 개행까지 싹 날림 (실무에서 가장 많이 씀)
        2. 루프로 제거: `while (cin.get() != '\n');`

```cpp
cin >> c1; // abc
// 입력 버퍼 상태는             [a][b][c][\n]
// `a`를 읽어서 `c1`에 저장     [b][c][\n] 버퍼에 그대로 남아 있는다
cout << c1 << endl; //        a 출력 97
cout << "next: " << cin.peek() << endl; // peek() 다음 문자 미리 보기
```
