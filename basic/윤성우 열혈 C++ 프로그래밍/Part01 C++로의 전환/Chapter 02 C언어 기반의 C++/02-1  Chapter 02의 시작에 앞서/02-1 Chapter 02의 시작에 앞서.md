## C언어의 복습을 유도하는 확인 학습 문제 1

### [문제 1] 키워드 const의 의미

> 키워드 const는 어떠한 의미를 갖는가? 다음 문장들을 대상으로 이를 설명해보자

#### const 키워드 쉽게 이해하기

> const는 `자물쇠` 이다

```cpp
const int num = 10;     // 자물쇠 잠금
num = 20                // 에러 - 못 바꿈
```

#### 포인터란?

> 포인터는 `다른 상자가 어디 있는지 주소를 담는 상자`

- 포인터로 할 수 있는 것
    - 포인터가 가리키는 방향을 바꾸기
    - 포인터로 상자 안의 값을 바꾸기

`[val1 = 5] <== [ptr1 = &val1]`
