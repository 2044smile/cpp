# cpp

이 저장소는 C++을 공부하고 실습하기 위해 만든 레포지토리입니다.

C++의 기본 문법부터 메모리 관리, 객체지향 프로그래밍 등 다양한 개념을 직접 코드로 구현하며 학습하는 것을 목표로 합니다.

또한 개인적으로 관심 있는 주제나 실험적인 코드들을 작성하면서 C++에 대한 이해를 높이려고 합니다.

## 🚀 C++을 배우는 이유

현재 Python을 사용하여 영상 처리와 데이터 처리 등의 간단한 테스트는 완료했지만, 더 높은 성능이 필요한 프로그램을 구현하고 시스템 수준의 동작을 이해하기 위해 C++을 배우고 있습니다. 

특히 영상 처리, AI 엔진, GPU 연산 등 성능이 중요한 분야에서는 C++이 많이 사용되기 때문에, Python에서 사용하고 있는 라이브러리들이 내부적으로 C++로 어떻게 구현되어 있는지 이해하는 것도 목표 중 하나입니다.
